Proyectos

Realicé un videojuego de plataformas en 2d con niveles sencillos en Gdevelop, no fué muy complicado, pero permitio que desarrollara más mi creatividad.

En el segundo videojuego trabajé en equipo y fue parecido al primer proyecto ya que trabajamos con 2D de Unity, pero ya requería un poco más de dificultad tanto en creatividad, entretenimiento y la estructura, además de que realizamos documentación al respecto.

En mi tercer proyecto realice un videojuego en 3D con Unity, en este punto ya era un trabajo un poco más complejo ya que se tenia que progamar ciertos items con java, el personaje o los enemigos y el daño asi como el diseño del juego.

En este proyecto realicé una pequeña base de datos en MySQL, se requirio programación con comandos de SQL además del uso de servidores como xamp y Querys para filtrar información y no represento mucha complejidad, también realicé documentación.

Trabajé con bases de datos no relacionales como Mongo DB, lo complejo de aqui es aprender los comandos que se necesitan para la creación de cada base de datos, realicé diferentes practicas y como proyecto de entrega hice una base de datos de una biblioteca.

También realice diferentes paginas web de diferentes tipos, algunas sobre productos y otra sobre promocion de turismo haciendo uso principalmente de html, css, bootstrap y java, lo interesante de esto es comprender la estructura de una página y ser creativo en lo visual.
